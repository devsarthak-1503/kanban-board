# Kanban Board (Drag & Drop)

A modern and responsive **Kanban Task Management App** built using **HTML, CSS and JavaScript**.

This app supports:
- ✔ Add New Tasks  
- ✔ Drag & Drop between Columns  
- ✔ Delete Tasks  
- ✔ Auto Task Counter  
- ✔ Smooth UI Animations  
- ✔ Fully Responsive Layout  
- ✔ LocalStorage Support (Tasks stay saved after reload)

---

## Features

### Add Tasks  
Add tasks with title & description from a modal popup.

### Drag & Drop  
Move tasks between:
- **To Do**
- **In Progress**
- **Done**

Drag & Drop is smooth and has placeholder animation.

### LocalStorage  
All tasks are permanently saved and automatically loaded when the page reloads.

### Responsive UI  
Looks great on:
- Mobile  
- Tablet  
- Desktop

## How to Use

1. Clone or download the project  
2. Open `index.html` in your browser  
3. Start adding tasks  
4. Drag tasks between columns  
5. Reload the page — your tasks will still be there
